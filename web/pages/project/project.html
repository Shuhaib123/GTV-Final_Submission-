What GTV does

Project GTV

GTV instruments Go programs, converts runtime trace output into a structured event stream, and renders an interactive topology graph. It is designed for education: stable layouts, step-by-step playback, and clear message-flow edges.

Modes
Offline parsing (trace.json) and Live streaming (WebSocket).
Entities
Goroutines as nodes, channels as rectangles, edges as message flow.
Export
PNG export + layout persistence per browser.
Tip: Use the Demo page to validate your UI quickly before running new workloads.
