How to use GTV

            
    
              
  • Choose a workload name, then click Run Once to generate a fresh trace.
    • 
              
  • Click Load Offline Graph to inspect the full run after it finishes.
    • 
              
  • Open Live Graph to watch events stream while a workload is active.
    • 
              
  • Use Sync Capture Preset for mutex/rwmutex/wg/cond-focused traces.
    • 
              
  • In Live, use Sync View to switch to debug events + causal overlay.
    • 
              
  • Optional: enable layer filters with ?layers=1 to toggle Channel/Sync/Spawn/Causal visibility.
    • 
              
  • Click any node or edge in a graph to see details, timing, and related events.
    • 
              
  • Use Legend, Fit View, and Clear Layout whenever the graph feels crowded.
    • 
            

            
Advanced view options like Split P2P and causal overlays are optional and can be explored later.
