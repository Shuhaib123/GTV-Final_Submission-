Go Trace Visualizer

Live Visualizer

Streams events over WebSocket from the Go demo workload.
Server flags: -addr :8080, -synth, -drop-block-no-ch
Env defaults: GTV_ADDR, GTV_SYNTH_SEND, GTV_DROP_BLOCK_NO_CH 
Examples:
go run ./cmd/gtv-live -addr :9090 -synth -drop-block-no-ch
GTV_SYNTH_SEND=1 go run ./cmd/gtv-live

Offline Visualizer

Load trace.json produced by the offline parser.
Offline:
go run .        # generates trace.out → trace.json
open graph.html # load the JSON in the page

Instrument & Run

Paste a Go main program, generate a workload, and run once to get JSON.
